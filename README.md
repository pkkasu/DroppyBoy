# Droppyboy 

[Droppyboy Live Site](https://droppyboy.herokuapp.com/)

## Background and Overview
Droppyboy is a clone of Dropbox.

## Features Overview

* User Auth/Account creation
* File Upload

Backend
* PostgreSQL
* Node.js

Frontend
* React
* Redux
* Javascript
